# Authentication App

## Overview
This Flutter authentication app uses SQLite for local storage and Firebase for managing favorite data. It includes user registration, login, and CRUD operations for handling authentication and favorite items.

## Features
- User registration and login
- Secure password hashing
- Local storage using SQLite
- Firebase integration for favorites
- CRUD operations for user and favorite data

## Technologies Used
- Flutter
- Dart
- SQLite
- Firebase Authentication
- GetX for state management
